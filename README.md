5、饿团外卖系统_数据库设计说明书.doc

**一、系统介绍**

可以注册登录账号

可以浏览店铺以及店铺内产品

可以添加产品进入购物车

结算时可以输入收货地址并且登记进入账号，有对应判空提示

可以后台管理店铺
